Lynis

Se lanza con el siguiente comando:
-lynis audit system
Si queremos hacer una auditoría de el sistema desde la cuenta -root:
-lynis audit system --pentest

Si no tenemos permisos de administrador y queremos hacer un test de penetración para descubir que puede encontrar un atacante desde una cuenta no privilegiada debemos lanza Lynis.
Lynis se puede ejecutar sin instalarlo.
